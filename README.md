## JEI Area Fixer
provides patches to fix GUI overlapping issues of JEI, with a config for enabling/disabling patches targeting specific mods.

Currently(version 1.2.0) supports:
- XNet
- RFTools
- mods made by OpenMods, like OpenBlocks
